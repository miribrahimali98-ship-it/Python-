## Ecommerce Analytics Project

### Objective
Analyze ecommerce sales data using SQL executed via Python to
understand customer behavior, order patterns, and product performance.

### Tools
- Python (pandas, matplotlib, seaborn)
- SQL (MySQL)
- mysql-connector

### Analysis Performed
- Average products per order by city
- Monthly order trends
- Product category sales analysis
- Customer purchase behavior analysis
- Moving average of customer payments
